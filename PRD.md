# PRD: Starter Builder (Task Manager Edition)

## 🎯 Visão Geral
Uma ferramenta interativa para alunos da Imersão IA personalizarem um "Gestor de Tarefas" e gerarem um prompt "premium" para Lovable/Google AI Studio, garantindo um resultado visual brutalista/moderno superior ao padrão.

## 👥 Público-Alvo
Alunos que querem construir a sua primeira app funcional com design profissional sem dominar CSS/UI complexo.

## 🚀 Funcionalidades (MVP)

### 1. Configurador de Design (Vibe Selector)
- **Estilo:** Brutalista (Dark/Light), Minimalista, Cyber-Gastro (Neon).
- **Cores Próprias:** Seleção de cor primária (Accent).
- **Tipografia:** Serifado (Elegante) vs Sans-Serif (Moderno).

### 2. Seletor de Funcionalidades (Logic Selector)
- **Modos de Visualização:** Lista Simples, Kanban (Colunas), Calendário.
- **Extras:** Prioridades (Baixa, Média, Alta), Categorias Customizáveis, Drag-and-Drop.

### 3. Motor de Geração de Prompt (Premium Prompt Engine)
- Transforma as escolhas num prompt estruturado.
- Inclui:
    - Stack Técnica (React + Tailwind + Lucide).
    - Diretrizes de Design Brutalista (bordas grossas, sombras duras, tipografia bold).
    - Lógica de Persistência (LocalStorage por defeito).

## 🛠️ Stack Técnica Sugerida
- **Frontend:** React + Vite + TypeScript.
- **Estilo:** CSS Inline / Tailwind (via CDN para rapidez).
- **Ícones:** Lucide React.

## 📅 Roadmap
- **Fase 1:** Mockup da Interface de Configuração.
- **Fase 2:** Engenharia do Prompt "Premium".
- **Fase 3:** Lógica de Cópia e Instruções de Uso.
