# Estratégia de Prompt: Starter Builder

## 🧱 O Conceito "Brutalista Moderno"
O diferencial desta ferramenta é garantir que o AI (Lovable/Gemini) não entregue um design genérico "Material Design" ou "Shadcn standard".

### Elementos Críticos do Prompt:
1. **Contraste Extremo:** "Usa #000000 puro para o fundo e #FFFFFF para o texto, com um accent color vibrante (#EEFF00)."
2. **Bordas e Sombras:** "Usa bordas sólidas de 2px e sombras duras (box-shadow: 4px 4px 0px #000) sem blur."
3. **Tipografia:** "Fonte sans-serif ultra-bold para títulos e mono para dados técnicos."
4. **Espaçamento:** "Paddings generosos e layouts assimétricos."

## 🧩 Estrutura do Prompt Gerado
O prompt final será dividido em:
- **[CONTEXTO]:** "És um Developer Senior especializado em UI Brutalista Moderno..."
- **[AÇÃO]:** "Cria um Gestor de Tarefas com as seguintes características..."
- **[DESIGN]:** Instruções específicas de CSS/Tailwind.
- **[LOGICA]:** Funcionalidades escolhidas pelo aluno.
- **[REGRAS]:** "Não uses gradientes, não uses bordas arredondadas suaves, mantém tudo RAW e INDUSTRIAL."
